# Contract Accounting API

`Contract Accounting API` — это минимально жизнеспособный продукт (MVP) бэкенда для учета договоров и взаиморасчетов с контрагентами (физическими и юридическими лицами).

Проще говоря, это серверная часть («мозг») для будущего сервиса, который поможет индивидуальным предпринимателям и малым компаниям вести учет финансов по договорам, отслеживать оплаты и контролировать дебиторскую задолженность, не прибегая к сложным бухгалтерским системам или Excel-таблицам.

## Стек технологий

- **Язык и фреймворк**: Python + FastAPI
- **База данных**: PostgreSQL
- **Инфраструктура**: Docker + Docker Compose
- **Миграции**: Alembic

## Запуск проекта

### Через Docker

```bash
docker compose up --build
```

После успешного запуска:

- **API** будет доступно по адресу: `http://localhost:8000`
- **Интерактивная документация (Swagger UI)**: `http://localhost:8000/docs`

## Миграции базы данных

Миграции применяются автоматически при старте контейнеров (через Alembic), поэтому дополнительной ручной настройки не требуется.

## Структура проекта

```text
app/
├── models/       # Модели базы данных
├── schemas/      # Pydantic-схемы (валидация данных)
├── routes/       # Эндпоинты API (FastAPI)
├── services/     # Бизнес-логика
└── migrations/   # Миграции Alembic
```
