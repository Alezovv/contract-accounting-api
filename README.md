# Contract Accounting API

Contract Accounting API — это минимально жизнеспособный продукт (MVP) бэкенда для учета договоров и взаиморасчетов с контрагентами (физическими и юридическими лицами).

Простыми словами: это серверная часть («мозг») для будущего сервиса, который поможет индивидуальным предпринимателям или малым компаниям вести учет своих финансов по договорам, отслеживать оплаты и контролировать дебиторскую задолженность, не прибегая к сложным бухгалтерским системам или Excel-таблицам.

## Стек технологий

- Python + FastAPI
- PostgreSQL
- Docker + Docker Compose
- Alembic

## Запуск проекта

### С помощью Docker

```bash
docker compose up --build
```

После запуска:

- **API** будет доступно по адресу: `http://localhost:8000`
- **Документация Swagger**: `http://localhost:8000/docs`

### Миграции базы данных

Миграции применяются автоматически при старте контейнера (через Alembic).  
Ручного вмешательства не требуется.

## Структура проекта

```
app/
├── models/       # Модели базы данных
├── schemas/      # Pydantic-схемы (валидация данных)
├── routes/       # Эндпоинты API (FastAPI)
├── services/     # Бизнес-логика
└── migrations/   # Миграции Alembic
```
